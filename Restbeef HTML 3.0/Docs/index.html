HTML/CSS Structure

This template is a responsive layout with sidebar. The general template structure is the same throughout the template. If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

  1. Open css/theme.css
  2. Find needed css-block.
  3. Change Value.

For easy navigation of the CSS file we placed Master Stylesheet in the beginning.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Edit Header

Editing the header is possible if you find a tag header.

This way you will have access to edit the logo and menu. To change the logo, upload the desired image to the img/logo.png folder and change size it to your own (theme.css, wherever you find a mention logo.png).
